# 餐厅订餐页

一个简单的静态前端订餐页面，包含菜品列表、购物车、总价计算和提交提示。

## 文件结构

```text
restaurant-ordering/
  index.html
  styles.css
  app.js
```

## 本地运行

直接双击 `index.html` 打开即可。

也可以使用 Node 启动本地静态服务：

```bash
npm run serve
```

然后访问输出的本地地址。

## Git 初始化

如果本机已安装 Git，可以在当前目录执行：

```bash
git init
git add .
git commit -m "初始化餐厅订餐页面"
```
